# Find-trip

Une application React moderne pour trouver votre prochaine destination de voyage.

## Description

Find-trip est une application web développée avec React et Vite qui aide les utilisateurs à découvrir et planifier leurs voyages. Que vous cherchiez une escapade urbaine, une aventure en nature ou des vacances relaxantes, Find-trip vous propose des suggestions personnalisées basées sur vos préférences.

## Fonctionnalités

- Recherche de destinations par critères (budget, durée, type de voyage)
- Suggestions personnalisées
- Interface utilisateur intuitive et responsive
- Intégration avec des APIs de voyage (à venir)

## Technologies utilisées

- **React** : Bibliothèque JavaScript pour construire des interfaces utilisateur
- **Vite** : Outil de build rapide pour le développement moderne
- **ESLint** : Outil de linting pour maintenir la qualité du code

## Installation

1. Clonez le repository :
   ```bash
   git clone https://github.com/LPAntoine/Find-trip.git
   cd Find-trip
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

4. Ouvrez votre navigateur à l'adresse `http://localhost:5173`

## Scripts disponibles

- `npm run dev` : Lance le serveur de développement
- `npm run build` : Construit l'application pour la production
- `npm run lint` : Vérifie le code avec ESLint
- `npm run preview` : Prévisualise la version de production
